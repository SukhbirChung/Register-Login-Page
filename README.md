"Register-Login Page" is a text based project that creates a register and login page.

When a user registers for a new account, there are certain requirements for each user input.

Requirements for First Name and Last Name:
These can't be empty.

Requirements for an Email:
There must be an @ sign in the email provided by the user.

Requirements for a Username:
Username can't be empty.
Username will not be accepted if it is already taken by someone else.

Requirements for a Password:
Password must be at least 8 characters long.
It can't be longer than 10 characters.
It must have at least one digit.
It must have at least one lowercase letter.
It must have at least one uppercase letter.

Confirmed password must be same as the password.
